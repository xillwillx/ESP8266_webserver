# ESP8266_webserver
Using an ESP8266-03 to turn a LED on/off with some jquery webcode.

Uses GPIO13 to control the LED

For flashing the ESP8266-03 you need to:
-Pull CH_PD to HIGH
-GPIO15 and GPIO0 to LOW
